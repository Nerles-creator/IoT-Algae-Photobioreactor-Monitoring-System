# IoT-Algae-Photobioreactor-Monitoring-System
IoT system to monitor and automate an algae cultivation system, using 2 microcontrollers (Mega and esp32), 6 sensors (pH, turbid, nutrient, tempt, light, co2), and motor pumps as actuators.
